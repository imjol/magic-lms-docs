# Page Setup

📄 Page Setup – Customize Your LMS Layout

The Page Setup feature in magicLMS allows admins to create, edit, and customize various pages within the platform. Easily design content-rich pages, adjust layouts, and personalize text and images to align with your branding. This feature ensures a fully customizable and user-friendly learning environment. 🎨✨

- **Click on Pages**
:nuxt-img{alt="page setup tab" src="https://cdn.imjol.com/MagicLMS/Docs/pages%20setup/Step1.png"}


- **Click on Page Create**
:nuxt-img{alt="page setup tab" src="https://cdn.imjol.com/MagicLMS/Docs/pages%20setup/Step2.png"}


- **Type "Daily Technology"**
:nuxt-img{alt="page setup tab" src="https://cdn.imjol.com/MagicLMS/Docs/pages%20setup/Step3.png"}


- **Click on Save**
:nuxt-img{alt="page setup tab" src="https://cdn.imjol.com/MagicLMS/Docs/pages%20setup/Step4.png"}


- **Click on Toggle to publish it**
:nuxt-img{alt="page setup tab" src="https://cdn.imjol.com/MagicLMS/Docs/pages%20setup/Step5.png"}
