# Application Setup Guide

***

### Minimum requirements check

- **Visit your custom domain in any web browser.**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step1.png"}


- **Click on Start Installation Process**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step2.png"}


- **Check if PHP Version (8.1+) is available**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step3.png"}


- **Check if Curl is Enabled**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step4.png"}


- **Check if .env File Permission is enabled**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step5.png"}


- **Check if Symlink is Enabled**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step6.png"}

### Connecting database with domain

- **Click on Go To Next Step**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step7.png"}

- **Type "127.0.0.1" in Database Hostname field**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step8.png"}


- **Type "your Database name" into Database name field**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step9.png"}


- **Type "your Database Username" into Database username field**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step10.png"}


- **Type "your Database password" in Database password field**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step11.png"}


- **Click on Save The Configuration**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step12.png"}


### Setting up organization page


- **Either click on import database (if you have already a database ready) or click on "demo sql". I'm moving with "demo sql"**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step13.png"}


- **Setting up organization setting**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step14.png"}


- **You can change organization logo, footer logo, header logo, soical handle link, phone number etc from this page. After all changes complete click on Save**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step15.png"}


### First time User setup

- **Type your desired Username**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step16.png"}


- **Type your valid Email address**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step17.png"}


- **Create a strong password for you user account**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step18.png"}

- **Type your password again and click on Register**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step19.png"}

- **Click on "Start Using Now" & it will redirect you to your new LMS homepage**
:nuxt-img{alt="apllication setup page via user domain" src="https://cdn.imjol.com/MagicLMS/Docs/application-setup/Step20.png"}
