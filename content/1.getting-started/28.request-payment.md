# Request Payment

💰 Request Payment (Instructor) – Easy Earnings Withdrawal

The Request Payment feature in magicLMS allows instructors to request payouts for their earned revenue seamlessly. Instructors can track their earnings, submit withdrawal requests, and monitor payment statuses from their dashboard. Admins can review and process payments efficiently, ensuring a smooth and transparent transaction experience. 🚀💵


- **Click on Request Payment**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step1.png"}


- **Click on Payment Request**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step2.png"}


- **Type "withdrawler amount"**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step3.png"}


- **Click on Select The Payment Method**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step4.png"}


- **Click on Bank**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step5.png"}


- **Type "a short description regarding the withdrawl"**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step6.png"}


- **Click on Save**
:nuxt-img{alt="payment request window" src="https://cdn.imjol.com/MagicLMS/Docs/payment%20request%20process/Step7.png"}
