# User Management 

Effortlessly manage admins, instructors, and students with dedicated panels in magicLMS. Assign roles, track progress, and maintain control—all in one intuitive platform! 👥

### Admin


- **Click on User Management**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step1.png"}


- **Click on Admins**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step2.png"}


- **Click on Add New Admin User**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step3.png"}


- **Type "yourusername"**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step4.png"}


- **Type "yourmailaddress"**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step5.png"}


- **Type your password**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step6.png"}


- **Click on Save**
:nuxt-img{alt="admin user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/admin/Step7.png"}

### Instructor


- **Click on User Management**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step1.png"}


- **Click on Instructors**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step2.png"}


- **Click on Add New Instructor**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step3.png"}


- **Select from the available packages**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step4.png"}


- **Type "your desired name"**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step5.png"}


- **Type "your mail address"**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step6.png"}


- **Type your password**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step7.png"}


- **Confirm your password**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step8.png"}


- **Click on Register account**
:nuxt-img{alt="Instructor user register panel" src="https://cdn.imjol.com/MagicLMS/Docs/user-managment/instructor/Step9.png"}
