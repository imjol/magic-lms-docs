# Quiz Setup

📝 Quiz Setup – Create Engaging Assessments

The Quiz Setup feature in magicLMS allows instructors to design and customize quizzes to assess learner understanding effectively. Set different question types, define time limits, enable automated grading, and customize quiz rules to match your course objectives. This feature ensures interactive and engaging assessments for a better learning experience. 🎯✅

- **Click on Quiz**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step1.png"}


- **Click on Quiz Create**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step2.png"}


- **Type "Wordpress short quiz"**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step3.png"}


- **Type "quiz duration"**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step4.png"}


- **Select a course which the quiz resides**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step5.png"}


- **Type "total marks"**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step6.png"}


- **Click on Submit**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step7.png"}


- **Click on Quiz List**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step8.png"}


- **Click on dropdown trigger**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step9.png"}


- **Click on Edit to edit the quiz content**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step10.png"}


- **Click on Update**
:nuxt-img{alt="quiz setup from instructor end" src="https://cdn.imjol.com/MagicLMS/Docs/Quiz/Step11.png"}
